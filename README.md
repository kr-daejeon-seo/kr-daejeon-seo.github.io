# 서구 · 대전광역시

> 서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daejeon-seo.github.io/**

서구는 대전광역시 중서부에 자리한 자치구로, 대전에서 인구가 가장 많은 최대 자치구다. 1985년 택지개발지구로 지정된 둔산지구가 1990년대 대규모 신도심으로 조성되면서 대전광역시청·정부대전청사·대전지방법원·대전지방검찰청 등 핵심 행정기관이 이곳으로 이전해 명실상부한 대전의 행정 중심지로 자리 잡았다. 둔산동 일대에는 갤러리아 타임월드를 비롯한 중부권 최대 규모의 상업·업무지구가 형성되어 있고, 한밭수목원과 둔산대공원, 대전예술의전당·대전시립미술관·이응노미술관이 한데 모여 문화예술 벨트를 이룬다. 구의 한복판을 갑천이, 동쪽 경계를 유등천이 흐르며, 남서쪽으로는 구봉산·장태산 등 산지가 이어져 도심과 자연이 공존한다. 면적은 약 95.48㎢이며, 24개 행정동(27개 법정동)으로 이루어져 있다.
